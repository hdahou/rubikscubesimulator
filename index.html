How to use the Rubik's Cube Console Simulator

  • Open the console and enter "cube" to view the initial cube array
  • Choose a move using the instructions below to print the new cube array...
    • Front Face Clockwise = f(cube)
    • Front Face Anti-Clockwise = fi(cube)
    • Right Face Clockwise = r(cube)
    • Right Face Anti-Clockwise = ri(cube)
    • Up Face Clockwise = u(cube)
    • Up Face Anti-Clockwise = ui(cube)
    • Back Face Clockwise = b(cube)
    • Back Face Anti-Clockwise bi(cube)
    • Left Side Clockwise = l(cube)
    • Left Side Anti-Clockwise = li(cube)
    • Down Face Clockwise = d(cube)
    • Down Face Anti-Clockwise = di(cube)
    • Reset = reset(cube)

Please note: The up face is array 0, left face array 1, front face array 2, right face array 3, back face array 4 and down face array 5.

You can access any particular face by entering in the console "cube[]" or any particular cubie by entering in the console "cube[][]" and this will print the cubie colour.
